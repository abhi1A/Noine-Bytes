This tool provides Bloom's taxonomy analysis for given file.
File must be in .txt format and in English
Punctuation and common words are removed during processing.

By level 1 meant querries:
Knowledge : write, list, label, name, state, define
Comperhension : explain, summarize, paraphrase, describe, illustrate
Application : use, compute, solve, demonstrate, apply, construct
Analysis : analyze, categorize, compare, contrast , separate
Synthesis : create, design, hypothesize, invent, develop
Evaluation : judge, recommend, critique, justify


By level 2 meant querries:
Knowledge : select, outline, write, list, recite, state, name, repeat, identify, locate, label, draw , record
Comperhension : explain, describe, relate, estimate, paraphrase, confirm, convert, match, compare, discuss, predict, infer
Application : construct, apply, modify, build, prioritize, report, sketch, produce
Analysis : analyze, categorize, compare, sort, investigate, debate, differentiate, examine
Synthesis : compose, combine, design, generate, hypothesize, formulate, originate, invent, revise
Evaluation : judge, justify, conclude, prioritize, rate, criticize, assess, appraise, critique
